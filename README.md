# evtolpy: Design and Simulation of eVTOL Aircraft and Missions

The software in this repository simulates characteristics of electric vertical
takeoff and landing (eVTOL) aircraft through mission segments to determine range
and estimate battery capacity.

## Directory Contents

* [analysis](analysis/README.md): Analysis scripts
* [docs](docs/README.md): Documentation
* [evtol](evtol/README.md): The Python package
* [references](references/README.md): Reference documents
* [sample-inputs](sample-inputs/README.md): Contains sample inputs for testing,
  demonstrations, and experiments
* [tests](tests/README.md): Package unit tests
* [README.md](README.md): This document
