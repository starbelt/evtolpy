# Maximum Range Analysis Results

This directory contains summarized maximum-range analysis results for the
AIAA SciTech 2026 paper  
**“Autonomous Battery Units as an Enabling Technology for Urban Air Mobility.”**

## Directory Contents

* [`maximum-range_1500_3000_ft.xlsx`](maximum-range_1500_3000_ft.xlsx): Summary analysis of maximum achievable mission range across all case studies
* [`cfg`](cfg): Directory containing all aircraft configuration JSON files used for the maximum range analysis
* [README.md](README.md): This document
