# ABU Analysis Results

This directory contains summarized analysis results related to the use of
Autonomous Battery Units (ABUs) for the AIAA SciTech 2026 paper  
**“Autonomous Battery Units as an Enabling Technology for Urban Air Mobility.”**

## Directory Contents

* [`detach_after_accelerate_climb_1500_3000_ft.xlsx`](detach_after_accelerate_climb_1500_3000_ft.xlsx): Assisted-takeoff ABU analysis results
* [`flight_extension_detach_on_depletion_or_end_1500_3000_ft.xlsx`](flight_extension_detach_on_depletion_or_end_1500_3000_ft.xlsx): Assisted-cruise ABU analysis results
* [`landing_safety_loiter_1500_3000_ft.xlsx`](landing_safety_loiter_1500_3000_ft.xlsx): Assisted-landing ABU analysis results
* [`8hr_common_case_economics_overlap_charging_queuing_1500_3000_ft.xlsx`](8hr_common_case_economics_overlap_charging_queuing_1500_3000_ft.xlsx): Effect of ABU usage on eVTOL flight cadence over an 8-hour period
* [`24hr_common_case_economics_overlap_charging_queuing_1500_3000_ft.xlsx`](24hr_common_case_economics_overlap_charging_queuing_1500_3000_ft.xlsx): Effect of ABU usage on eVTOL flight cadence over a 24-hour period
